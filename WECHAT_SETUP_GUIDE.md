# 微信开放平台配置保姆级教程 (Android)

想要实现“跳转微信登录”，你必须通过微信官方的审核。这是一个**严格**且**必须**的流程。请按照以下步骤一步步操作：

## 第一阶段：注册与认证 (电脑操作)

1.  **访问官网**
    打开 [微信开放平台 (open.weixin.qq.com)](https://open.weixin.qq.com/)。

2.  **注册账号**
    *   点击右上角“注册”。
    *   **注意**：这与你的个人微信号、公众号都不同，是一个全新的**开发者账号**。你需要用一个新的邮箱注册。

3.  **开发者资质认证 (必做)**
    *   注册成功登录后，点击顶部“账号中心” -> “开发者资质认证”。
    *   **关键点**：个人开发者无法申请“微信登录”接口！**你必须拥有一个企业营业执照**（或者个体工商户）。
    *   **费用**：腾讯收取 **300元/年** 的审核费。
    *   按照提示上传营业执照、法人身份证等信息，等待审核通过（通常 1-3 个工作日）。

---

## 第二阶段：创建移动应用

认证通过后，我们开始创建这个 APP 的“身份证”。

1.  点击顶部菜单 **“管理中心”** -> **“移动应用”** -> **“创建移动应用”**。
2.  **填写基本信息**：
    *   **应用名称**：慈
    *   **应用简介**：记录情侣甜蜜时光的专属APP。
    *   **应用图标**：上传项目的 `public/favicon.svg` (转成 28x28 和 108x108 的 PNG 格式)。
3.  **填写平台信息 (Android)**：
    *   勾选 **Android 应用**。
    *   **应用官网**：如果没有官网，可以先填一个简单的页面地址或者暂填 `https://www.baidu.com` (后续可改)。
    *   **应用包名**：`com.cjyh.app` (这是我们项目中固定的 ID，千万别填错)。
    *   **应用签名 (最关键一步)**：
        微信通过“包名+签名”来识别你的 APP。你需要填写正确的 **MD5 签名**。
        *   **如何获取调试版签名**：
            在你的电脑终端中运行以下命令（假设你安装了 JDK）：
            ```bash
            keytool -list -v -keystore "%USERPROFILE%\.android\debug.keystore" -alias androiddebugkey -storepass android -keypass android
            ```
            *   在输出信息中找到 **MD5** 这一行，去掉冒号，把那一串字母数字变成**小写**（例如 `5e8f1b...`），填进去。
4.  **提交审核**：
    *   提交后等待微信审核（通常 1-7 个工作日）。

---

## 第三阶段：获取 AppID 并填入代码

审核通过后：

1.  回到 **“管理中心”** -> **“移动应用”**，点击你刚创建的应用。
2.  你会看到一个 **AppID** (以 `wx` 开头) 和 **AppSecret**。
3.  **申请接口**：在应用详情页左侧，找到“微信登录”接口，点击“申请开通”。

### 最后一步：回到本项目配置

打开项目根目录下的 `capacitor.config.ts` 文件，将你的 AppID 填入：

```typescript
const config: CapacitorConfig = {
  // ... 其他配置
  plugins: {
    Wechat: {
      appId: "wx1234567890abcdef" // <--- 把这里换成你申请到的 AppID
    }
  }
};
```

**重新打包**：
每次修改配置后，都需要运行以下命令生效：
```bash
npx cap sync
npx cap open android
```
然后在 Android Studio 中重新点击 Build APK。

---

## 常见坑点 ⚠️

1.  **签名不匹配**：
    *   我们在开发时使用的是 `debug.keystore`（通用调试签名）。
    *   **正式发布时**（上架应用商店），你会生成一个正式的 `.jks` 密钥文件。
    *   **切记**：正式版的签名和调试版不一样！**上线前**记得去微信后台，把“应用签名”修改为正式版 keystore 的 MD5 签名。

2.  **包名错误**：
    *   一定要确保微信后台填写的包名是 `com.cjyh.app`。

祝你一次过审！🚀
